一个U盘，一台电脑，怎么重装系统

发表于2022-08-17|更新于2022-08-18|教程
|字数总计:501|阅读时长:1分钟|阅读量:

如何重装系统

下载安装微PE

  • 官网：微PE工具箱 - 超好用的装机维护工具 (wepe.com.cn)
  • 下载安装到电脑上

下载win10镜像

  • 网站：MSDN -> 操作系统

  • 选择相应系统

  • 下载迅雷

  • 复制磁力链接到迅雷下载

下载激活工具HEU_KMS_Activator

  • 官网：Releases · zbezj/HEU_KMS_Activator (github.com)

微pe制作启动盘制作工具

  • U盘8G以上，建议16G

  • 插入u盘

  • 把U盘上的东西存于电脑上可找见位置

  • 打开微PE

  • 选择安装pe到U盘

  • 格式化模式选NTFS -> 立即安装到U盘 -> 开始制作

  • 上一步会格式化U盘，所以有东西要提前拿出存于电脑上

  • 结果：会分为大小两个盘

  • U盘启动盘做好

备份电脑文件

  • 备份个人重要文件进入启动盘中大的盘
  • 备份镜像、激活工具等进大的盘

进入PE系统

  • 关机

  • 插入系统启动盘

  • 开机

  • 不停按对应按键进入BIOS

  • 上下左右移动选项（每个电脑的BIOS界面不同）

  • 进入Exit

  • 选择BOOT Override -> U盘启动盘第二个选项（小的那个盘） ->Enter选择进入PE系统

格式化磁盘

  • 打开我的电脑

  • 右键对应磁盘格式化该磁盘

分区助手对硬盘进行分区

  • 点击硬盘这几个字选中硬盘

  • 点击快速分区

  • 设置分区数目

  • 设置分区大小

  • 设置扇区：2048改为4096

  • 点击执行

windows安装器

  • 选择Windows安装文件的位置 -> 搜索

  • 选择大的盘中的镜像

  • 选择安装驱动的位置 -> 搜索

  • 选择系统盘C盘

  • 选项 -> 选择对应的系统版本

  • 安装 -> 确定 -> 确定

  • 弹出U盘

  • 重启

激活系统

  • 插入启动盘

  • 打开激活工具

  • 智能激活

  • 开始

本教程借鉴网络教程，纯属个人总结，若有相似，纯属雷同。
