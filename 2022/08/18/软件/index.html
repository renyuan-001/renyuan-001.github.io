收藏的软件

桌面美化

  • 雨滴（桌面美化插件）

  • clickshow（显示鼠标点击效果）

  • 7tsp GUI（更换系统图标）

  • UltraUXThemePatcher（系统主题卸载）

  • TranslucentTB（透明化状态栏）

  • mydesk（桌面dock栏）

  • Tile Genie（开始菜单个性化磁贴）

电脑相关

  • geek（软件卸载）

  • IDM（下载工具）

  • FDM（下载工具）

  • Traffic Monitor（显示当前网速、CPU及内存利用率）

  • Windows Update Blocker（开启关闭系统自动更新）

  • Dism++（c盘清理）

  • Mem Reduct（自动清理内存）

  • spacesniffer（磁盘空间分析工具）

  • Everything（文件搜索工具）

  • SoftCnKiller（清理流氓软件和流氓文件）

  • PotPlayer（网络播放器）

  • Quicker（效率软件）

  • utools（插件化、跨平台的现代桌面软件）

Windows激活

  • KMSAuto
  • HEU_KMS_Activator

系统重装

  • 微PE

markdown编辑器

  • typora

  • typedown

编程软件

  • Eclipse（java）

  • IDEA（java）

  • Vscode

  • Navicat（MySQL）

  • Git

  • Notepad++

安全

  • 火绒安全

压缩

  • Winrar

  • 360压缩

时间管理

  • 滴答清单

  • 时光序

不翻墙GitHub

  • Watt Toolkit

  • GreenHub

图床

  • PicGo
