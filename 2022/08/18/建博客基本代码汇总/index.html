搭建博客可能会用到的命令

个人搭建博客常用命令

顺序不分先后，当作工具书用

node.js和npm安装

node -v（查看node.js版本、cmd）

npm -v（查看npm版本、cmd）

npm config get prefix（查看安装目录、cmd）

npm config get cache（查看缓存目录、cmd）

npm root –location=global（查看包的存放路径、cmd）

npm config set prefix “D:\studio\nodejs\node_global”（指定安装目录、cmd）

npm config set cache “D:\studio\nodejs\node_cache”（指定缓存目录、cmd）

git安装

git –version（查看git版本）

安装hexo

npm i hexo-cli –golbal=location（博客目录、git控制台）

hexo -v（查看版本）

卸载hexo

npm uninstall hexo-cli –location=global(git控制台)

生成 ssh keys

ssh（任意文件夹、git控制台）

ssh-keygen -t rsa -C “你注册GitHub时的邮件地址”（生成 ssh 命令、总共需要敲四次回车）

cat ~/.ssh/id_rsa.pub（找到 id_rsa.pub 文件代码）

本地博客生成

hexo init（初始化博客命令、git控制台、博客目录）

hexo s（开启本地服务的命令、git控制台、博客目录）

部署博客到 github

npm install hexo-deployer-git –save（安装 hexo-deployer-git 自动部署发布工具、git控制台、博客目录）

hexo g（生成页面、git控制台、博客目录）

git config –global user.name “GitHub用户名”（修改用户名、git控制台、博客目录）

git config –global user.email “GitHub注册邮箱@qq.com”（修改邮箱、git控制台、博客目录）

hexo d（本地文件上传到 Github 上面、git控制台、博客目录）

Git 版本的控制

git init（生成本地仓库命令、git控制台、博客目录）

git add .（将项目添加到暂存区、git控制台、博客目录）

git commit -m “注释内容”（将项目提交到Git仓库、git控制台、博客目录）

git branch -M main（上传到 main 分支、git控制台、博客目录）

git remote add origin https://github.com/renyuan-001/blog-backup.git（和远程仓库连接、git控制台、博客目录）

git push -u origin main（ 将本地项目推送到远程仓库、git控制台、博客目录）

git pull –rebase origin main（跟刚创建的线上远程仓库的默认分支main关联、git控制台、博客目录）

克隆到本地

git init（将本地仓库初始化、git控制台、要克隆到的目录）

git clone url（url为为项目服务器地址或github地址、git控制台、要克隆到的目录）

安装Butterfly

npm i hexo-theme-butterfly（使用 npm 的方式，在 vscode 终端安装、Hexo 根目录里、git控制台）

npm update hexo-theme-butterfly（升级方法、 Hexo 根目录下、git控制台）

npm install hexo-renderer-pug hexo-renderer-stylus –save（安装 pug 以及 stylus 的渲染器、根目录、git控制台）

新建标签页

hexo new page tags（博客目录、git控制台）

新建分类页

hexo new page categories（博客目录、git控制台）

新建友情链接

hexo new page link（博客目录、git控制台）

新建图库

hexo n page xxxxx（博客目录、git控制台）

安装Share.js

npm install social-share.js(博客目录、git控制台)

清除缓存文件 db.json 和已生成的静态文件 public

hexo clean

安装hexo-algoliasearch

npm install hexo-algoliasearch –save

安装字数统计

npm install hexo-wordcount –save

看板娘

npm install –save hexo-helper-live2d（安装插件、博客目录、git控制台）

hexo clean

hexo g

hexo s

更换看板娘

npm install –save live2d-widget-model-hibiki（更换、博客目录、git控制台）

npm install –save live2d-widget-model-hijiki（更换、博客目录、git控制台）

上传数据到algolia

hexo algolia（博客目录、git控制台）


收纳不全，持续更新。
